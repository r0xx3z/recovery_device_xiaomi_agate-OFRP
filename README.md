# Device specifications 

| Device                  | Xiaomi 11T                                  |
| ----------------------- | :---------------------------------------------------------- |
| SoC     		  | MediaTek Dimensity 1200-Ultra (MT6893)                    |
| CPU     		  | Octa-core (1x 3.0 GHz Cortex-A78, 3x 2.6 GHz Cortex-A78, 4x2.0 GHz Cortex-A55) |
| OS Version  | MIUI 12.5 (Android 11) to hyperOs 1 (Android 14) |
| GPU     		  | Mali-G77 MC9                      |
| Memory      |  8GB RAM                  |
| Storage     | 128GB / 256GB                  |
| Battery 		  | 5000 mAh, Li-Polymer |
| Display 		  | AMOLED, 120Hz, HDR10+, Dolby Vision, 6.67 inches, 1080 x 2400 pixels, 20:9 (~395 ppi), ~800 nits (typ) |
